## 🌟 Обо мне

Привет! Я **DANIEL45690** (известный как **concole_hack**) из Самары.  
Обожаю превращать идеи в рабочие инструменты: от крутых консольных утилит и GUI-приложений до полноценных веб-сервисов.

Люблю Python, PyQt5, веб-технологии и всё, что можно "взломать" в хорошем смысле 😉

- **Сейчас активно развиваю** инструменты для Windows/Linux
